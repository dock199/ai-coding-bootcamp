## 決定事項

- **技術スタック:** Python (FastAPI) + React + Azure OpenAI GPT-4.1
- **API Framework:** FastAPI
- **デプロイ環境:** Azure App Service（FastAPIデプロイ）
- **開発期間:** 2週間でMVP完成
- **データベース:** 使用しない（セッションベースの一時保存のみ）
- **環境変数管理:** `.env`ファイル + `python-dotenv`を使用。`.env`は`.gitignore`対象。`.env.example`をリポジトリに含める。本番環境はAzure App Serviceのアプリケーション設定を使用
- **日報形式:** 「作業内容」「進捗状況」「課題・問題点」の3セクション
- **対応ブラウザ:** モダンブラウザ（IE11非対応）

## アクションアイテム

- **佐藤:** React環境セットアップ、UI/UX作成
